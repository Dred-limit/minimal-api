# Fruit Store API

Простой и современный REST API для управления данными, разработанный на ASP.NET Core Minimal API. Этот проект демонстрирует практическое применение ключевых концепций бэкенд-разработки на .NET.

## Технологии и подход

*   **Платформа**: ASP.NET Core 8 (Minimal API)
*   **Язык**: C# 13.0
*   **Хранение данных**: Потокобезопасная коллекция `ConcurrentDictionary` (имитация работы с БД)
*   **Ключевые концепции**: Dependency Injection, Endpoint Filters, валидация моделей, типизированные результаты (`TypedResults`), работа с `LinkGenerator` для генерации URI.
*   **Стиль**: Чистый, документированный код с использованием современных возможностей .NET 8.

## Функционал API (CRUD)

| Метод  Эндпоинт  Описание 

| `GET`  `/fruit`  Получить список всех фруктов. 
| `GET`  `/fruit/{id}`  Получить фрукт по ID. Валидация: ID должен начинаться с 'f'. 
| `POST`  `/fruit/{id}`  Добавить новый фрукт. Возвращает заголовок `Location` со ссылкой на созданный ресурс. 
| `PUT`  `/fruit/{id}`  Полностью обновить данные фрукта. 
| `DELETE` `/fruit/{id}`  Удалить фрукт по ID. 

## Как запустить проект

1.  **Клонируйте репозиторий:**
    ```bash
    git clone [https://github.com/Dred-limit/minimal-api.git]
    cd minimal-api
    ```
2.  **Запустите приложение:**
    ```bash
    dotnet run
    ```
3.  **API будет доступен по адресу:** `https://localhost:[PORT]` (порт будет указан в консоли).
